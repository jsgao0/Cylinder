# Cylinder
Draw cylinders as SVG using jQuery

## Requirement
1. SVG support
2. jQuery 1.6+

## Usage
```
$('targetElement').cylinder({
  height: 300, // Cylinder height.
  width: 70, // Cylinder width.
  color: {
    head: '#669900', // The color of the top of the cylinder.
    body: '#88cc00' // Surrounding color of the cylinder.
  }
});
```
